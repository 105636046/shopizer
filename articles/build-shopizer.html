Build Shopizer


Shopizer is an e-commerce application framework built in Java. The source code has to be built into binary format and package as a war (web archive) file prior to be deployed to any Java application server. Shopizer uses Maven as build management tool which requires to be installed in order to complete the steps described below. (See softeware requirements section). The software can be build standalone or from your favorite IDE tool that must be integrated to Maven.

- Standalone build with Maven

- Build from inside Eclipse

Standalond build with Maven


$ cd shopizer
$ mvn clean install

Build from inside Eclipse


- Shopizer project needs to be imported as maven project. In the workspace you should see shopizer parent module along with all other shopizer child modules.
- Right click on shopizer parent project, select maven clean
- Right click on shopizer parent project, select maven install
