Get Shopizer


Shopizer is available on Git Hub. You can clone the software at the command line or download the zip file.

Get shopizer using git command line


You must have a git client installed on your workstation in order to perform this task. 

   $ git clone https://github.com/shopizer-ecommerce/shopizer.git

This will clone the default Shopizer branch


Download Shopizer zip file from Git Hub


Click on download from Shopizer Git Hub repository https://github.com/shopizer-ecommerce/shopizer
